A jar file was used to avoid having to compile 60+ classes one by one.
The bat file must be in the same folder as the jar file.

How to run:
- Open the bat file in the folder "Maze_jar"

How to play:
- Arrow keys/WASD for 4 directional movement
- L to attack with auto aim
- Mouse click to attack with manual aim
- Bow and throwing projectiles (bombs) attack on release, all other weapons attack on press
- Goal is to reach the end*

* End condition is not implemented yet
* Enemies seem to be broken
* Spears have not been added yet
